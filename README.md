# Metronome

it clicks

## Technologies

- **React 19**
- **Vite**
- **Tailwind CSS**
- **Lucide React Icons**

## Getting Started

### Prerequisites
- Node.js (v16 or higher)

### Installation

```bash
npm install
```

### Development

```bash
npx vite dev
```

Starts the development server at `http://localhost:5173`

### Build

```bash
npx vite build
```

Creates an optimized production build in the `dist/` directory.
